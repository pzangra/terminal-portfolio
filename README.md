# TerminalPortfolioOS: Dual-Access Terminal Portfolio

A character-locked personal viewport for Pietro Maria Zangrando.

## Technical Specs
- **Architecture:** Fixed-grid 130ch x 37.5em container using pure CSS ch units.
- **Rendering:** Optimized for 1:1 parity between w3m/lynx and modern browsers.
- **Visuals:** #00ff00 Phosphor Green, CRT Scanline Overlay, Monospace only.
- **Deployment:** GitHub Pages (Web) / SSH-Visitor Shell (Future).

## Objective
Bridging Biomedical Engineering research with terminal-centric developer workflows.
